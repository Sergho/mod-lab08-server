Было исследовано поведение многоканальной СМО с отказами с различными параметрами
В качестве изменяемого параметра была интенсивность входного потока заявок
Для каждого набора параметров были вычислены основные показатели СМО двумя способами:
1. Фактически - путём симуляции работы СМО
2. Теоретически - путём использования формул
Полученные данные представлены в текстовом виде и в виде графиков

Из графиков видно, что по все пары показателей коррелируют
Корреляция имеет слабый характер по причине малого количетсва отправляемых заявок при тестировании
Кроме этого по каждому из графиков в отдельности можно сделать вывод о поведении каждого из параметров:
- Вероятность простоя системы уменьшается с ростом интенсивности входного потока заявок, что логично
- Напротив, вероятность отказа системы растёт вместе с ростом интенсивности входного потока
- Относительная пропускная способность ведёт себя полностью противоположно вероятности отказа системы
- Абсолютная пропускная способность подобно относительной растёт с ростом интенсивности входного потока
- Наконец, среднее число занятых каналов растёт с ростом интенсивности входного потока, приближаясь к своему максимуму, равному числу доступных каналов

Указанные замечания верны при фиксированной интенсивности потока обслуживания, причём под ростом интенсивности входного потока подразумевается её рост относительно интенсивности потока обслуживания
